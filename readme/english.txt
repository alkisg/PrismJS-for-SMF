[color=green]This script highlights syntax in the code blocks on SMF forums.[/color][/center]

[list]
[li]It is necessary to specify the language used: [nobbc][code=html]<h1>Hello world</h1>[/code][/nobbc].[/li]
[li]You can specify the default language and then short syntax is available: [nobbc][code]echo 'How are you?';[/code][/nobbc][/li]
[li]The theme style can be changed (8 styles).[/li]
[/list]
